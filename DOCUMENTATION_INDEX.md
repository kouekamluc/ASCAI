All diagrams use **Mermaid** syntax, which is supported by:
- GitHub (native)
- GitLab (native)
- VS Code (with extension)
- Many markdown viewers
- Online tools (mermaid.live)

### Viewing Diagrams
1. **GitHub/GitLab**: View directly in repository
2. **VS Code**: Install "Markdown Preview Mermaid Support"
3. **Online**: Copy diagram code to [mermaid.live](https://mermaid.live)
4. **Documentation Sites**: Most static site generators support Mermaid
